Welcome to Stargaze!

This is the web interface for controlling the Stargaze star pointer. Send your position to the server controlled laser pointer and select whether to track Stellarium© or your location.
location_onSend Location
my_locationTrack Satelite
star_rateUse Stellarium
power_settings_newStand By
About Stargazer

Stargazer is a project created by Jonathan Juursema, Guido Kuijper, Rzonnie Jonkman & Peter Verzijl. It is a colaboration project between Computer Science and Electrical Engineering students. This project is its result.
